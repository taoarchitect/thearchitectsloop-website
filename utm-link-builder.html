    Frequently asked questions

    Does this tool send my URL anywhere?

    No. The link is built entirely in your browser.

    Do I need Google Analytics to use UTMs?

    No. UTMs are just URL parameters. They are useful when your site or landing page already uses an analytics product that reads them.

    What if my URL already has UTM parameters?

    This builder removes existing utm_* parameters on the same URL, then applies the values you enter above.
